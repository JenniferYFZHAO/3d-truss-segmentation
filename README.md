# 3D 桁架结构点云分割与配准工具

## 项目结构

```
3d-truss-segmentation/
├── main.py             # 主分割工具 - 基于模板信息的点云分割
├── register.py         # 配准工具 - OBJ 模型与 LAS 点云配准
├── utilities/          # 辅助工具和功能模块
│   ├── easy_surface_extract.py      # 简单表面提取示例
│   ├── generation.py                # 桁架结构点云生成
│   ├── possion_surface_extraction.py # Poisson 表面重建
│   ├── read_cloudpoints.py          # 点云读取和处理
│   ├── segmentation.py              # 初步分割算法
│   └── visualization.py             # 点云可视化
└── README.md           # 项目说明
```

## 主要功能

### 1. 主分割工具 (`main.py`)
- **功能**：基于结构模型的模板信息（球节点和圆管）对 LAS 点云进行分割
- **输入**：
  - CSV 格式的模型信息文件（包含球节点和圆管的位置和尺寸）
  - TXT 格式的变换矩阵文件（包含 OBJ 中心化矩阵和 ICP 精配准矩阵）
  - LAS/PLY 格式的点云文件
- **输出**：
  - PLY 格式的分割结果（带颜色编码）
  - CSV 格式的标签文件
- **特点**：
  - 向量化计算，提高处理速度
  - 为每个球节点和圆管分配唯一标签
  - 支持点云下采样，提高处理效率

### 2. 配准工具 (`register.py`)
- **功能**：将 OBJ 模型与 LAS 点云进行配准
- **步骤**：
  1. 读取 OBJ 模型和 LAS 点云
  2. 交互式选取控制点
  3. 计算相似变换（旋转 + 平移 + 缩放）
  4. 执行 Similarity ICP 精配准
  5. 可视化配准结果
- **特点**：
  - 支持相似变换（包含缩放）
  - 从 mesh 表面采样点作为目标，提高配准精度
  - 动态调整对应点距离阈值

### 3. 辅助工具 (`utilities/`)

#### 3.1 点云读取 (`read_cloudpoints.py`)
- 支持读取 LAS/LAZ/PLY 格式的点云文件
- 支持大文件处理和进度显示
- 提供点云下采样、质心计算、平移等功能

#### 3.2 点云生成 (`generation.py`)
- 生成带球节点的圆管桁架点云
- 支持添加噪声和噪声点
- 用于测试和算法验证

#### 3.3 表面重建 (`possion_surface_extraction.py`)
- 基于 Poisson 算法的表面重建
- 包含点云预处理、法线估计等步骤
- 支持保存为 PLY/OBJ/STL 格式

#### 3.4 简单表面提取 (`easy_surface_extract.py`)
- 基于生成的桁架点云进行表面提取示例
- 调整参数以适应桁架结构

#### 3.5 初步分割 (`segmentation.py`)
- 基于节点坐标和构件连接信息的粗分割算法
- 用于测试分割思路

#### 3.6 可视化 (`visualization.py`)
- 使用 Matplotlib 进行 3D 点云可视化
- 支持按构件 ID 着色

## 使用方法

### 1. 配准流程
1. 运行 `register.py`
2. 按照提示选取控制点
3. 观察粗配准和 ICP 精配准结果
4. 保存配准后的点云和变换矩阵

### 2. 分割流程
1. 确保已有配准后的点云文件和变换矩阵文件
2. 运行 `main.py`
3. 程序会自动读取模型信息、应用变换、分割点云
4. 可视化分割结果并保存

## 注意事项

1. **文件路径**：请在代码中修改默认文件路径以适应你的项目
2. **参数调整**：
   - 点云下采样因子：根据点云大小调整
   - 分割阈值：根据点云密度和模型精度调整
   - ICP 参数：根据配准效果调整
3. **性能优化**：
   - 对于大文件，建议使用较大的下采样因子
   - 对于复杂模型，可能需要调整 ICP 的迭代次数和收敛阈值
